# Website

## Stack

- [Next.js](https://nextjs.org/) - The React Framework
- [React](https://ja.reactjs.org/) - Javascript library for build UI.

## Command

run:

```shell
npm run dev
# ready - started server on 0.0.0.0:3000, url: http://localhost:3000
```

Pretty the source code:

```shell
npm run pretty
```
